What is Kubeflow?

Kubeflow 项目致力于将机器学习 (ML) 部署到 Kubernetes 上简单化、可移植且可扩展。 我们的 目标不是重新创建其他服务，而是提供一种直接的方式来将 用于 ML 的同类最佳开源系统 部署到不同的基础设施。 在任何运行 Kubernetes 的地方，都应该能够运行 Kubeflow。

Notebooks

Kubeflow 包括创建和管理交互式的 Jupyter notebooks。您可以自定义笔记本部署和计算资源 以满足您的数据科学需求。在本地试验您的工作流程， 然后在您准备好后将它们部署到云中。

TensorFlow 模型训练

Kubeflow 提供了一个自定义的 TensorFlow 训练任务控制器 你可以用来训练你自己的 ML 模型。通常 Kubeflow 的 job operator 可以处理分布式 TensorFlow 训练任务。 将训练控制器配置为使用 CPU 或 GPU 并适应各种集群大小。

模型服务

Kubeflow 支持 TensorFlow Serving 容器将经过训练的 TensorFlow 模型导出到 Kubernetes。 Kubeflow 还集成了 Seldon Core，在 Kubernetes 上部署机器学习模型的 开源平台，在大规模部署 ML/DL 模型时最大化 GPU 利用率的 NVIDIA Triton Inference Server 以及 MLRun Serving，用于部署和监控的开源无服务器框架。


工作流

Kubeflow 工作流 Pipeline是部署和管理端到端ML工作流的综合解决方案。 使用Kubeflow管道进行快速可靠的实验。 您可以安排和比较运行，并检查每次运行的详细报告。

多框架支持

我们的发展计划超越了TensorFlow。我们正在工作 难以扩展的支持 PyTorch, Apache MXNet, MPI, XGBoost, Chainer 等等。我们 也集成了 Istio 及 Ambassador 流量入口， Nuclio 作为一个 快速多用途服务网格框架，并使用 Pachyderm 管理 你的数据实验工作流。
