*experiment* 是一个工作区，您可以在其中尝试不同的管道配置。
您可以使用实验将运行组织到逻辑组中。
实验可以包含任意运行，包括
[recurring runs](/docs/components/pipelines/concepts/run/)。

* 阅读 [Kubeflow Pipelines 概述](/docs/components/pipelines/introduction/).

  部署 Kubeflow 并直接从 Kubeflow Pipelines UI 运行示例管道。
